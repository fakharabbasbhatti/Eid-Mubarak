HTML 3D Animated سوالات اور جوابات

1. HTML کیا ہے؟
HTML (HyperText Markup Language) ایک زبان ہے جو ویب پیجز بنانے کے لیے استعمال ہوتی ہے۔
2. HTML ٹیگ کیا ہوتا ہے؟
HTML ٹیگ زاویہ دار بریکٹ < > میں لکھا جاتا ہے اور ویب پیج کے عناصر کو ظاہر کرتا ہے۔ مثال: <p>یہ ایک پیراگراف ہے</p>
3. HTML اور CSS میں کیا فرق ہے؟
HTML ویب پیج کی ساخت بناتا ہے جبکہ CSS ویب پیج کو خوبصورت بنانے کے لیے استعمال ہوتا ہے۔
4. HTML دستاویز کی بنیادی ساخت کیا ہوتی ہے؟
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
یہاں مواد لکھا جاتا ہے
</body>
</html>
5. <!DOCTYPE> کا مقصد کیا ہے؟
یہ براؤزر کو بتاتا ہے کہ HTML کا کون سا ورژن استعمال ہو رہا ہے۔
6. <head> ٹیگ کیا ہے؟
<head> میں ویب پیج کی معلومات ہوتی ہیں جیسے عنوان، CSS فائلیں اور اسکرپٹ۔
7. <title> ٹیگ کیا کرتا ہے؟
<title> ویب پیج کا عنوان ظاہر کرتا ہے جو براؤزر کے ٹیب میں نظر آتا ہے۔
8. <body> ٹیگ کیا ہے؟
<body> میں وہ تمام مواد ہوتا ہے جو ویب پیج پر نظر آتا ہے جیسے متن، تصاویر اور لنکس۔
9. HTML Attributes کیا ہوتے ہیں؟
Attributes ٹیگ کے بارے میں اضافی معلومات دیتے ہیں۔ مثال: <a href="https://google.com">Google</a> یہاں href attribute ہے۔
10. <div> اور <span> میں کیا فرق ہے؟
<div> بلاک لیول عنصر ہے جبکہ <span> ان لائن عنصر ہے۔
11. HTML میں Hyperlink کیا ہے؟
Hyperlink ایک ویب پیج کو دوسرے پیج سے جوڑتا ہے۔ مثال: <a href="https://google.com">Google پر جائیں</a>
12. <img> ٹیگ کس لیے استعمال ہوتا ہے؟
<img> ویب پیج پر تصویر دکھانے کے لیے استعمال ہوتا ہے۔ مثال: <img src="image.jpg" alt="تصویر">
13. HTML Lists کیا ہیں؟
Lists چیزوں کو ترتیب سے ظاہر کرنے کے لیے استعمال ہوتی ہیں۔ اقسام: Ordered List <ol> اور Unordered List <ul>۔
14. <br> ٹیگ کیا کرتا ہے؟
<br> نئی لائن بنانے کے لیے استعمال ہوتا ہے۔
15. <hr> ٹیگ کیا کرتا ہے؟
<hr> افقی لائن بنانے کے لیے استعمال ہوتا ہے۔
16. HTML Form کیا ہے؟
Form صارف سے معلومات لینے کے لیے استعمال ہوتا ہے۔ مثال: <form><input type="text"><input type="submit"></form>
17. <table> ٹیگ کیا ہے؟
<table> ٹیگ ٹیبل بنانے کے لیے استعمال ہوتا ہے۔ مثال: <table><tr><th>Name</th></tr></table>
18. <b> اور <strong> میں فرق کیا ہے؟
<b> صرف متن کو موٹا کرتا ہے جبکہ <strong> اہم متن کو ظاہر کرتا ہے۔
19. <meta> ٹیگ کیا ہے؟
<meta> ویب پیج کے بارے میں معلومات فراہم کرتا ہے جیسے description اور charset۔
20. Semantic HTML Elements کیا ہیں؟
وہ ٹیگز جو اپنے مطلب کو واضح کرتے ہیں۔ مثال: <header>, <footer>, <section>, <article>۔
21. HTML میں <iframe> کیا ہے؟
<iframe> ایک ویب پیج کے اندر دوسرا ویب پیج دکھانے کے لیے استعمال ہوتا ہے۔
22. HTML میں <audio> اور <video> ٹیگز کیا ہیں؟
<audio> آڈیو چلانے کے لیے اور <video> ویڈیو چلانے کے لیے استعمال ہوتے ہیں۔
23. HTML Entities کیا ہیں؟
HTML Entities خصوصی علامات کو ظاہر کرنے کے لیے استعمال ہوتی ہیں، جیسے &lt; <، &gt; >۔
24. HTML میں Comments کیسے لکھتے ہیں؟
HTML میں Comments اس طرح لکھتے ہیں: <!-- یہ ایک comment ہے -->۔ یہ براؤزر میں نہیں دکھائی دیتا۔
25. HTML میں <link> ٹیگ کیا ہے؟
<link> ٹیگ ویب پیج میں CSS فائل یا favicon شامل کرنے کے لیے استعمال ہوتا ہے۔
26. HTML میں <form> کے مختلف input types کیا ہیں؟
Common input types: text, password, email, number, submit, radio, checkbox, file وغیرہ۔
27. HTML میں <nav> ٹیگ کیا ہے؟
<nav> ویب پیج میں نیویگیشن لنکس کے لیے استعمال ہوتا ہے۔
28. HTML میں <section> اور <article> میں کیا فرق ہے؟
<section> ایک ویب پیج کا سیکشن ہے جبکہ <article> آزاد مواد یا مضمون کو ظاہر کرتا ہے۔
29. HTML میں <canvas> ٹیگ کیا ہے؟
<canvas> ٹیگ گرافکس یا animations بنانے کے لیے استعمال ہوتا ہے JavaScript کے ساتھ۔
30. HTML میں Responsive Design کے لیے کون سی meta tag استعمال ہوتی ہے؟
<meta name="viewport" content="width=device-width, initial-scale=1.0">۔ یہ موبائل ڈیوائس پر ویب پیج کو درست دکھانے کے لیے ضروری ہے۔
