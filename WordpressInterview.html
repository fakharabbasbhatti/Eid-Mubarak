🚀 WordPress Interview Questions

1. What is WordPress?
WordPress is an open-source Content Management System (CMS) used to create websites and blogs easily.
2. What are WordPress Plugins?
Plugins are tools that extend the functionality of a WordPress website such as SEO, security, and contact forms.
3. What is a WordPress Theme?
A theme controls the design, layout, and appearance of a WordPress website.
4. What is a Child Theme?
A child theme inherits the functionality of the parent theme and allows customization without modifying the original theme files.
5. What are WordPress Hooks?
Hooks allow developers to add or modify functionality in WordPress without changing the core files.
